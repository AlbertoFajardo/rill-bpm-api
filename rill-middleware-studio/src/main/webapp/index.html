【工作流引擎】Rill-BPM - 轻量级企业BPM解决方案

基于Activiti、Tomcat、Embedded Glassfish构建企业级工作流平台.

企业级

平台基于轻量级Servlet容器 - Tomcat6构建独立的工作流平台，通过Web Service可与业务系统在分布环境中进行事务协调。

平台性

支持大规模的流程需求，服务端可以通过水平扩展（Web服务、DB服务）来增加平台吞吐能力。

客户端的事务可以通过Web Service流入到平台端。（如下图）

访问项目主页：http://code.google.com/p/rill-bpm-api/

使用的开源软件：

  • Activiti
  • Tomcat
  • Embedded Glassfish
  • Spring Framework
  • jQuery

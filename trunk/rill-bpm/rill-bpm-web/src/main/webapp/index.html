流程管理

这里是一个 流程管理 入口。流程部署、流程查询、创建流程等 进入

流程部署

提供web接口的流程部署功能，可以部署单个/多个（zip格式压缩）流程定义文件。

流程查询

提供正在运行中的流程查询功能。可以查看流程的详细信息，比如发起人、发起时间、当前任务等；也可以监控流程运行的轨迹。

创建流程

提供动态表单渲染功能。可以在部署流程后直接创建流程。
